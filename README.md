# Sugar Insight – Food Label Analyzer

Sugar Insight is an AI-powered web app that analyzes nutrition label images to detect sugar content.

It solves the problem of confusing and misleading food labels by converting them into simple insights.

Users can upload a product label and instantly see sugar levels (Low, Medium, High).

The app also detects hidden sugar ingredients from the ingredient list.

Built with React, TypeScript, Tailwind CSS, and AI/OCR integration.

Focuses on clean UI, fast performance, and real-world usability.

Includes features like visual sugar meter and ingredient highlighting.

Helps users make smarter and healthier food choices.

Designed as a production-ready frontend + AI project.
